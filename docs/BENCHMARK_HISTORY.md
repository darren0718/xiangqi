## v5-p4a-search1 (Step 9A — Phase B 第 3 步 · 搜索增强 Part 1)

- TT aging：TTEntry 新增 8-bit `age`；每次 `ai_move` `tt_age = tt_age.wrapping_add(1)`；替换策略去掉 rand，改为「空 / 旧 age / 深度更深 / 老条目是 UPPER」四选一
